# [liblapin-tutorial](https://liblapin.readthedocs.org/) #

An alternative tutorial for liblapin

## Dependencies ##

- python2
- sphinx
- sphinx-autobuild
- recommonmark

```sh
$ pip install sphinx sphinx-autobuild recommonmark
```

https://docs.readthedocs.org/en/latest/getting_started.html

## Build ##

```sh
$ make html
$ make examples
```
